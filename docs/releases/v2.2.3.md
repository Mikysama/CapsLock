# CapsLock v2.2.3

CapsLock 2.2.3 是 fullscreen TUI 的交互与透明渲染补丁。默认 inline 界面、公开 CLI、Agent 行为和持久化协议保持不变。

## Fullscreen 交互

- 修复从 Composer 提交 `/model` 后模型选择器卡死的问题。需要等待模态结果的斜杠命令现在由 Textual worker 执行，App 消息泵可以继续处理选择和关闭事件。
- 回归测试通过真实 Composer 输入、Enter 提交、方向键选择和再次 Enter 确认覆盖完整用户路径，不再绕过消息处理器直接调用内部命令方法。

## 透明渲染

- fullscreen App 根层使用终端原生 `ansi_default`，支持继承终端配色及终端模拟器自身的透明度；Screen、消息、输入区和模态容器继续透明叠加。
- Composer 的当前输入行、光标、光标 gutter 和括号匹配不再绘制黑色背景；光标改用下划线保持可见。
- 助手 Markdown 的行内代码、代码块及审批 Syntax 预览通过字符级渲染适配器清除背景。适配器只将背景替换为终端默认值，不改变前景色、粗体、暗化、斜体、下划线、删除线或链接。

## 兼容性

本版本不增加工具、权限、网络访问或持久化字段。workspace schema v5、memory schema v3、config v2、portable archive v2、JSONL v2、CLI 退出码和审批语义均保持不变。

## 验证

本地发布验证包含 Ruff format/check、repository hygiene、版本一致性、构建与 Twine 检查，以及 214 项测试。fullscreen 测试覆盖模态模型选择、终端默认根背景、Composer 输入、Markdown/Syntax 字符背景和字体样式保留。
