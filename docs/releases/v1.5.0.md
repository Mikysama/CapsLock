# CapsLock v1.5.0 未发布说明

v1.5.0 的开发提交没有创建 Git tag 或 GitHub Release。该版本试验过一套强类型 Skill 运行时，但在公开发布前已由 v1.5.1 的 Agent Skills 风格契约取代；CapsLock 不读取旧包，也不提供转换命令。

v1.5.0 中完成的 `.capslock/` 目录重构继续保留，但其 Skill 路径兼容、运行时、领域类型、CLI 查询和 audit 表均已删除。v1.5.1 将工作区 schema 升至 v5，并在自动备份后移除旧 `skill_runs` 数据。

请以 [v1.5.1 发布说明](v1.5.1.md) 作为 v1.5 系列的安装、Skill 编写和发布依据。不要创建 `v1.5.0` tag。
