# CapsLock v2.1.0

CapsLock 2.1.0 新增本地工具插件 SDK 与受控安装流程。插件从本地目录显式安装，逐工作区启用，并通过现有高风险外部动作审批和审计链调用。

## 新增

- `capslock.plugin_sdk` 和版本化 JSONL/stdio 协议。
- 本地插件安装、升级、校验、启用、禁用和卸载命令。
- 包内容 SHA-256、权限展示、工作区授权和用户级生命周期审计。
- 动态插件工具发现，以及超时、取消、输出上限和摘要变化保护。

## 安全边界

插件在独立子进程运行并只继承最小环境，但仍属于受信本地代码。2.1.0 不提供恶意代码沙箱、在线插件源、签名索引或市场。

升级后插件需要在工作区重新启用。`.capslock/local/plugins.json` 不进入 portable archive，恢复或迁移不会自动授权插件。

## 兼容性

最低 Python 版本仍为 3.12，支持 Linux 和 macOS。workspace schema v4、memory schema v3、portable archive v2、JSONL v2 和 config v2 保持不变。
